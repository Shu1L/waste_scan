## waste_scan

一个简单的web综合扫描器。

#### 起因

前一段时间读了《白帽子讲web扫描》这本书，因为书中代码是以python2写的。而且最终也没有一个能跑的项目。

  所以自己参考书上的代码，尝试用python3重构了下代码，主要学习怎么从0开始设计开发一款简单漏洞扫描器。
